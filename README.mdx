Hello and welcome to the repo for my solidjs website project.

![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/kurtmcmurt/kurtisrogers.com/node.js.yml)

If you want something a bit more bells and whistles for Solid (like Next or Nuxt), check out [Solid Start](https://docs.solidjs.com/solid-start)!


This project is using the ultra naked quickstart version which you can find [here](https://docs.solidjs.com/quick-start#for-typescript-projects). Enjoy!

> [!IMPORTANT]
> ## Node version
> This project contains an `.nvmrc` file so if you have that set up on your machine, you can simply run `nvm use`.
>
> **Recommended**
> ```sh
> v20.18.0
> ```
>
> ## Installation
> ```sh
> yarn install
> ```

> [!NOTE]
> ### Production mode (locally)
> ```sh
>   yarn build
>   yarn serve
> ```
